# Move Agent

## Prompt
Du bist ein erfahrener Python-Entwickler mit Expertise in agentenbasierten Systemen und dem Pydantic AI Framework.

Ich arbeite an einem Agenten-Projekt. Die Architektur und der Aufbau sind in den Dateien `agent_plan.md` und `doku_agent.md` beschrieben. Die aktuell geöffneten `.py`-Dateien enthalten die bisherige Implementierung. Meine Aufgabe ist in `agent_plan.md` definiert. 

**Deine Aufgabe:** Hilf mir beim vollständigen Aufbau des **Move Agents** und seiner zugehörigen Tools in Python.

Gehe dabei wie folgt vor:

1. Lies `agent_plan.md`, `student_taks_2.md` und `doku_agent.md`, um den Gesamtaufbau, die Rolle des Move Agents und die erwartete Tool-Struktur zu verstehen. 
2. Lies `student_task_2.md`, um meine konkrete Aufgabenstellung und Anforderungen zu verstehen.
3. Analysiere die bestehenden `.py`-Dateien, um Konventionen, Strukturen und bereits implementierte Patterns zu übernehmen.
4. Implementiere den Move Agent und seine Tools gemäß der Spezifikation aus den Dokumenten — orientiere dich dabei an den offiziellen Pydantic AI Core Concepts für Agenten: https://pydantic.dev/docs/ai/core-concepts/agent/

**Anforderungen an die Implementierung:**
- Folge exakt den Konventionen und Strukturen der bestehenden `.py`-Dateien im Projekt
- Implementiere alle für den Move Agent vorgesehenen Tools als eigenständige, typisierte Funktionen
- Verwende Pydantic AI korrekt (Agent-Definitionen, Tool-Dekoratoren, Typen)
- Stelle sicher, dass der Code produktionsreif ist: korrekte Fehlerbehandlung, klare Typisierung, konsistente Benennung
- Kommentiere kritische Stellen kurz auf Deutsch, konsistent mit dem Rest des Projekts 

Gib den vollständigen, ausführbaren Code aus — keine Platzhalter, keine Pseudocode-Abschnitte.
